# Dashboard
## Dashboard de visualização de dados em python com:
* DASH PLOTLY
* HTML
* CSS

Para rodar o projeto em sua máquina, siga os comandos:

```
$ git clone https://github.com/ErickBalbino/dashboard.git 
$ cd dashboard
$ python3 app.py
```
